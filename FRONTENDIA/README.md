`C:\Users\default.DESKTOP-CNVIP6E\Desktop\Proyectos\IA\FRONTENDIA`

## Instalacion

```bash
cd C:\Users\default.DESKTOP-CNVIP6E\Desktop\Proyectos\IA\FRONTENDIA
npm install
```

## Ejecucion

```bash
cd C:\Users\default.DESKTOP-CNVIP6E\Desktop\Proyectos\IA\FRONTENDIA
npm run dev
```

Abrir:

```text
http://localhost:5173
```
